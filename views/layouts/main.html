Scrum

Principales motivos para elegir la metodología SCRUM:

  • Usa sprints quincenales, que ayudan a llevar el proyecto al día.
  • Es muy adecuada para proyectos de tamaño medio como el nuestro.
  • Aporta flexibilidad, existe la posibilidad de cambiar de acuerdo a nuevas necesidades sin grandes problemas.
  • Funciona bien cuando el equipo de desarrollo no está familiarizado con las herramientas que va a utilizar.

    Tecnologías
  • Ejecución sobre un servidor web Tomcat embebido en SpringBoot. Para la implementación de los servicios utilizamos el framework Jersey, estando todo desarrollado en Java.
  • Diseño e implementación del cliente en Angular2, juntamente con los componentes del framework PrimeNG y los grids de Bootstrap para el posicionamiento y la parte responsive de la plataforma.
  • Componentes desarrollados en Typescript, un superconjunto de Javascript compilado que nos permite obtener una aplicacion más rápida.
